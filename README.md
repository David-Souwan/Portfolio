## Technologies

HTML5, CSS3 et JavaScript ES6, sans framework ni dépendance JavaScript.

> La police est chargée depuis Google Fonts. Pour un fonctionnement 100 % hors ligne, remplacez-la par des fichiers de police locaux dans `assets/`.

## Installation

1. Téléchargez ou clonez ce dossier.
2. Ouvrez `index.html` dans un navigateur moderne.
3. Pour le développement, utilisez idéalement une extension de serveur local (par exemple Live Server dans VS Code).

## Personnalisation

- Remplacez la photo dans `david.jpg` et adaptez son chemin dans `index.html` si nécessaire.
- Les études de cas se trouvent dans la section `#projets` de `index.html`.
- Les valeurs GitHub sont volontairement indicatives et s’actualisent via les attributs `data-count`.
- Modifiez les couleurs globales dans les variables en tête de `css/style.css`.
- Les interactions sont regroupées dans `js/script.js`.

## Déploiement sur GitHub Pages

1. Créez un dépôt GitHub et envoyez les fichiers du projet.
2. Dans le dépôt, ouvrez **Settings → Pages**.
3. Choisissez **Deploy from a branch**, puis la branche `main` et le dossier `/ (root)`.
4. Enregistrez : votre portfolio sera disponible via l’URL GitHub Pages indiquée.
